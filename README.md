# Excel Pivot-Table-Projects
# A collection of Excel Pivot Tables
This repository contains Excel Pivot Table projects demonstrating practical data analysis, reporting, and dashboard-building skills. Each project uses publicly available or course-generated datasets to showcase your ability to organize, summarize, and visualize data for insights.

US Voters’ Case Study (2012)
Dimensions: State, Age
Measures: Total Population, Citizen Population, Registered Voters, Confirmed Voters
Concepts Covered: Calculated Fields, % of Column, % of Parent
Skills Used: Pivot Tables
Outcome: Summarized voting populations by state and age, analyzed engagement trends, and calculated key voter statistics.

San Francisco Government Employee Salaries (2011–2013)
Dimensions: Employee Name, Year, Employee ID, Job Title
Measures: Base Pay, Overtime Pay, Other Pay
Concepts Covered: Sorting & Filtering, Grouping, Calculated Fields, Table Layouts
Skills Used: Pivot Tables
Outcome: Analyzed employee compensation trends over time, created structured reports for multiple years.

Shark Attack Records (1900–2016)
Dimensions: Case Number, Date, Type, Country, Area, Location, Activity, Name, Gender, Age, Injury, Fatal (Y/N), Species, Investigator/Source
Measures: None
Concepts Covered: Date Grouping, Value Settings (% of Column, Running Total), PivotCharts
Skills Used: Pivot Tables
Outcome: Explored shark attack trends over time and geography, using date grouping and pivot charts to identify patterns.

Stock Market Sample Data
Dimensions: Date, Symbol
Measures: Open, High, Low, Close, Volume
Concepts Covered: Sorting & Filtering, Conditional Formatting (% Difference From)
Skills Used: Pivot Tables
Outcome: Summarized stock performance for 500 publicly traded companies over 3 months, highlighting trends and differences.

Major League Baseball Statistics (1995–2015)
Dimensions: Season Start, Year, League, Team, Division
Measures: G, W, L, RS, AB, H, HR, ERA, etc.
Concepts Covered: Sorting & Filtering, PivotCharts & Slicers, Calculated Fields
Skills Used: Pivot Tables
Outcome: Analyzed team and league performance, created visual summaries with pivot charts and slicers.

Burrito Ratings & Yelp Reviews (65 Restaurants, San Diego, 2016)
Dimensions: Location, Burrito, Date, Recommendation, Reviewer, Notes
Measures: Yelp Rating, Cost, Tortilla, Temp, Fillings, Meat Volume, Uniformity, Salsa, Synergy, Wrap Quality
Concepts Covered: COUNT, SUM, AVERAGE, Rank, Conditional Formatting
Skills Used: Pivot Tables
Outcome: Provided insights into burrito quality and restaurant performance using calculated and aggregated metrics.

Boston Daily Weather (2016)
Dimensions: Date, Conditions
Measures: Max Temp, Mean Temp, Min Temp, Max Wind Speed, Precipitation
Concepts Covered: Sorting & Filtering, PivotCharts, Date Grouping, COUNT
Skills Used: Pivot Tables
Outcome: Summarized weather patterns over a year, visualizing seasonal trends in temperature, wind, and precipitation.

Spartan Race Facebook Posts (Aug–Oct 2016)
Dimensions: Page, Date of Post, Post Copy, Link, Post Type, Hour of Post, Time of Day
Measures: Shares, Reactions, Likes, Comments
Concepts Covered: Show Values As (% of Column), Calculated Fields, COUNT
Skills Used: Pivot Tables
Outcome: Analyzed engagement metrics by post type and timing, highlighting trends to optimize social media strategy.
